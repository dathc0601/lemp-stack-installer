# LEMP Stack Installer

One-command LEMP stack installer for Ubuntu 22.04 and 24.04.

Nginx (mainline) + MariaDB 11.4 + PHP 8.4 + Composer + Node.js 22 + Redis + Certbot + phpMyAdmin + File Browser

> **Under construction** — This project is being refactored from a monolithic script
> into a modular installer. See `server-setup.sh` in the repo root for the current
> working (v2.0.1) single-file version.
